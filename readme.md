# How to run

Step 1. Install uv
```
pip install uv
```

Step 2. Create virtual environment
```
uv venv
```

Step 3. Activate virtual environment
```
source .venv/bin/activate
```

Step 4. Install dependencies
```
uv pip install -r requirements.txt
```

Step 5. Run the program
```
python main.py
```
